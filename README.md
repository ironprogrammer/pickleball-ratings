# Pickleball Ratings

This plugin provides a customizable block to display player DUPR (Dynamic Universal Pickleball Rating) ratings information. It uses the official DUPR API and requires a DUPR account to authenticate API requests.

In addition to the plugin itself, the `docs/` directory includes the requirements and agent implementation instructions that were used to assist the development of the plugin. See the `.cursor/rules` directory for additional Cursor-specific rules. This plugin was designed and built with the assistance of Claude, Cursor, and cursor-agent.

## Development

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the plugin:
   ```bash
   npm run build
   ```
4. Install the ZIP or symlink and activate the plugin in WordPress.

Other development commands include:

- `npm run build` - Build the plugin for production
- `npm run start` - Start development mode with hot reloading
- `npm run clean` - Remove build files for fresh start
- `npm run lint` - Run all linters (PHP, CSS, and JS)
- `npm run lint:js` - Lint JavaScript files
- `npm run lint:css` - Lint CSS files
- `npm run lint:php` - Lint PHP files
- `npm run format` - Format code with Prettier and PHP CS Fixer
- `npm run format:php` - Format PHP code only
- `npm run test:setup` - One-time WordPress test suite setup (creates test DB and installs WP tests)
- `npm test` - Build assets and run PHPUnit tests
- `npm run packages-update` - Update WordPress packages
- `npm run plugin-zip` - Create production-ready plugin ZIP with full quality assurance (clean, lint, test, build, package, verify)

### Plugin Structure

```
pickleball-ratings/
├── src/                          # Source files
│   ├── index.js                  # Main Gutenberg block JavaScript
│   ├── frontend.js               # Frontend JavaScript for block functionality
│   ├── block.json                # Block configuration
│   └── style.css                 # Block styles
├── build/                        # Built files (generated by wp-scripts)
├── admin/
│   └── class-pbr-admin-settings.php # Admin settings page
├── includes/
│   ├── class-pbr-dupr-api.php    # DUPR API integration
│   └── class-pbr-ajax-handler.php # AJAX request handling
├── tests/                        # PHPUnit test files
├── pickleball-ratings.php        # Main plugin file
├── package.json                  # NPM dependencies and scripts
├── composer.json                 # PHP dependencies
└── readme.txt                    # WordPress.org plugin readme
```

### Testing

To run tests locally:

1. One-time setup (installs WP test suite and creates DB):
   ```bash
   npm run test:setup
   ```
2. Build assets and run PHPUnit:
   ```bash
   npm test
   ```
